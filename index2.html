REVERSE BREATHING

Reverse breathing is actually a very popular technique among many athletes to increase lung capacity, and strenghten the breathing system and the stomach area

Some key points to remeber while doing reverse breathing are:

  1. Do not do it if you are pregnant, there is surgery done recently and if you have critical stomach problems.
  2. It is best practiced empty stomach, it is not necessary but advised.
